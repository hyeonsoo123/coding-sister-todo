# 📅 프로젝트 작업 캘린더 - TODO 관리 앱

프로젝트의 작업을 캘린더 기반으로 효율적으로 관리할 수 있는 웹 애플리케이션입니다.
날짜별로 할 일을 등록하고, 우선순위로 관리하며, 진행 상황을 추적할 수 있습니다.

## ✨ 주요 기능

- **📅 캘린더 뷰**: 월별 캘린더에서 작업이 있는 날짜 한눈에 파악
- **✅ TODO 관리**: 제목, 설명, 우선순위와 함께 작업 등록
- **🎯 우선순위 설정**: 높음, 중간, 낮음 3단계 우선순위
- **📊 필터링**: 전체, 진행중, 완료 필터로 작업 관리
- **⏱️ 작업 추적**: 생성 시간, 완료 시간 자동 기록
- **📱 반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 최적화
- **💾 자동 저장**: 브라우저 localStorage로 데이터 자동 저장

## 🚀 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:8081` 열기

## 💻 기술 스택

- **HTML5**: 시멘틱 마크업
- **CSS3**: Tailwind CSS + 커스텀 반응형 스타일
- **JavaScript (ES6+)**: 캘린더와 TODO 관리 로직
- **LocalStorage**: 클라이언트 측 데이터 저장

## 📱 반응형 디자인

- ✅ **모바일 최적화**: 터치 친화적인 버튼 (44px 이상)
- ✅ **태블릿 최적화**: 2열 레이아웃
- ✅ **데스크톱**: 3열 레이아웃 (캘린더 + TODO)
- ✅ **가로/세로 회전 대응**: 자동 레이아웃 조정

## 🎯 사용 방법

### 1. 캘린더에서 날짜 선택
- 캘린더에서 원하는 날짜 클릭
- 클릭한 날짜의 작업만 오른쪽에 표시됨

### 2. 작업 추가
1. **작업 제목** 입력 (필수)
2. **상세 설명** 입력 (선택)
3. **우선순위** 선택 (높음/중간/낮음)
4. "➕ 작업 추가" 버튼 클릭
5. Enter 키로도 추가 가능

### 3. 작업 관리
- **완료 표시**: 체크박스 클릭 (완료/미완료 토글)
- **삭제**: "🗑️ 삭제" 버튼 클릭
- **필터링**: "전체", "진행중", "완료" 탭 클릭

### 4. 작업 정렬
- 우선순위 순으로 자동 정렬 (높음 > 중간 > 낮음)
- 미완료 작업이 완료된 작업보다 위에 표시

## 🎨 우선순위 색상

- 🔴 **높음** (빨강): 긴급 작업
- 🟡 **중간** (노랑): 보통 작업
- 🟢 **낮음** (초록): 여유 작업

## 💾 데이터 관리

모든 작업은 브라우저의 localStorage에 자동 저장됩니다.
- 데이터는 로컬 PC에만 저장됨
- 브라우저 캐시 삭제 시 데이터 손실 가능
- 백업이 필요한 경우 개발자 도구에서 데이터 내보내기 권장
