# Coding Sister Todo 프로젝트 개요

## 프로젝트 소개
개인 코딩 프로젝트 관리 및 TODO 리스트 애플리케이션

## 프로젝트 요구사항
- [ ] 요구사항 1 : HTML, CSS, Javascript 사용해서 포트폴리오 앱 사이트 제작.
- [ ] 요구사항 2 : 스타일링은 Tailwind CSS를 사용
- [ ] 요구사항 3 : 페이지 제작시 데스크톱과 모바일 모두에서 잘 보이는 반응형으로 적용. 
- [ ] 요구사항 4 : 모바일 관련 추가사항 - 터치 친화적인 버튼 크기(최소 44px), 가로/세로 화면 회전 대응  
모바일 키보드 고려한 레이아웃, 빠른 로딩속도.
- [ ] 요구사항 5 : Todo 앱 각 기능별로 파일을 나눠서 작성 - HTML, CSS, JavaScript 파일 분리, 컴포넌트별로 별도 파일 생성

## 주요 기능
- TODO 관리
- 프로젝트 트래킹
- 개인 노트 작성

## 기술 스택
- Frontend: JavaScript
- Backend: Java
- Database: MongoDB
